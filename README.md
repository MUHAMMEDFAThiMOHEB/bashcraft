# bashcraft

Repo contains my handmade applications and ideas made using BASH scripting during my journy of learning BASH

the repo contains some sub-directories for categorizing each type for more clarification

- [Backup_Scripts](#backup-scripts)
- [Calculators](#calculators)
- [DataEntry](#data-entry)
- [Games](#games)
- [Learning_Applications](#learning-applications)

## Backup Scripts

This directory / category contains different bash scripts,
each one is doing the same purpose which is archiving and compressing files in a given direcotry
each script is differ from the other in using keywords, commands, tools ...

## Calculators

Contains some scripts which share the functionality "do mathematical operations", with different methods
varity of BASH hacks, Arithmetic Expansion, Command Substituation, positional parameters, special parameters

## Data Entry

Contains scripts which main function is to get, display and modify data taken from user

## Games

let's have some fun with some simple games ideas performed using BASH scripting

## Learning Applications

The Core or the milestone dirctory of this repo, as its contains all the applications that I have build during my journey of learing bash scripting,
starting from what's BASH, going through Variables, Shell Expansions, Requesting user input
Logic, Conditions, Loops, Processing Options, Reading files and so on .....
